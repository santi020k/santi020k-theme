# Santi020k iTerm2 Theme Website

Static Astro product site for [iterm.santi020k.com](https://iterm.santi020k.com/). It previews and distributes the dark and light presets generated by `packages/santi020k-iterm-theme`.

```bash
pnpm run site:iterm:dev
pnpm run site:iterm:build
```

The build syncs generated `.itermcolors` files into `public/themes`. Edit the canonical palettes in the package, not the website copies.
