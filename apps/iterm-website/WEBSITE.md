# iTerm2 website operations

- Production domain: `https://iterm.santi020k.com/`
- Local port: `4176`
- Build output: `apps/iterm-website/dist`
- Canonical theme source: `packages/santi020k-iterm-theme/palettes.mjs`

Deploy this app as an independent static site. Configure the `iterm.santi020k.com` custom domain at the hosting provider and point its DNS record to that project. Theme downloads are copied into the app during every build.
