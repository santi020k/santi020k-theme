# Santi020k Terminal Theme website

Static Astro product site for [terminal.santi020k.com](https://terminal.santi020k.com/). It presents the complete Terminal product and publishes dedicated guides for iTerm2 and Starship.

```sh
pnpm run site:terminal:dev
pnpm run site:terminal:build
```

The build syncs generated presets from `packages/santi020k-terminal-theme`. Edit canonical palette and generator sources in the package, not website copies.
