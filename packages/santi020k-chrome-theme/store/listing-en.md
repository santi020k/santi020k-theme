## Long description (paste into "Detailed description" field)

```
Santi020k Theme brings the exact color palette from the popular Santi020k VS Code extension to your browser chrome.

Every surface maps directly to a VS Code token — no guesswork, no approximations:

• Toolbar and tab strip: #1c1528 (sideBar.background)
• Inactive tabs: #0b0712 (tab.inactiveBackground) — blend into the frame
• Active tab accent line: #752df0 (tab.activeBorder)
• New Tab page background: #110c1d (editor.background)
• Omnibox / address bar: #110c1d with #dfdde3 text
• Links and accents: #945df4 / #b48df7 (textLink.foreground)
• Incognito: a slightly deeper frame variant of the same palette

Works with all Chrome builds that support Manifest V3 themes. Load it as an unpacked extension in Developer Mode for instant preview, or install from the Chrome Web Store.

Source: https://github.com/santi020k/santi020k-theme/tree/main/packages/santi020k-chrome-theme
VS Code extension: https://marketplace.visualstudio.com/items?itemName=santi020k.santi020k-theme
```
