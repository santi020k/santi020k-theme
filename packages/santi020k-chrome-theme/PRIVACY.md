# Privacy Policy — Santi020k Theme (Chrome)

**Single purpose:** This extension changes the visual appearance of the Chrome browser (colors, tab strip, toolbar, and New Tab page) to match the Santi020k VS Code color theme. It performs no other function.

**Data collection:** None. The extension does not collect, store, transmit, or sell any user data — including browsing history, personal information, or usage analytics.

**Permissions:** This extension requests no browser permissions. Theme extensions operate entirely through the `theme` manifest key; no content scripts, background workers, or host permissions are used.

**Remote code:** None. The extension ships no scripts of any kind. All behavior is declarative JSON.

**Third parties:** No third-party libraries, services, or SDKs are included.

**Contact:** santi020k@gmail.com
