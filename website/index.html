Philosophy

Designed for hours, not screenshots.

Every decision in the palette came from a question: does this help you read code faster?

No neon

Saturated syntax colors destroy contrast at 2 AM. Every hue is pulled back until it guides the eye without demanding attention.

Keywords recede

Control-flow keywords are muted italic — present but quiet. Your identifiers and values carry the visual weight because they carry the logic.

Semantic-aware

Semantic highlighting is on by default. Class names, type parameters, and mutable variables get distinct shades — no extra config needed.

Terminal included

The 16 ANSI colors are calibrated to the same violet palette. Shell output reads like part of the editor, not an afterthought.

Works with

Every VS Code-based editor.

Install once from the Marketplace or Open VSX — the theme loads in all editors built on VS Code's extension API.
