# VS Code Theme Surface Checklist

Use this checklist when a theme surface looks unstyled or when expanding coverage.

## Core

- `foreground`, `descriptionForeground`, `errorForeground`, `icon.foreground`
- `focusBorder`, `selection.background`, `widget.shadow`
- `textLink.foreground`, `textLink.activeForeground`

## Widgets And Modals

- `editorWidget.background`, `editorWidget.foreground`, `editorWidget.border`, `editorWidget.resizeBorder`
- `input.background`, `input.foreground`, `input.border`, `input.placeholderForeground`
- `inputValidation.{info,warning,error}{Background,Foreground,Border}`
- `button.*`, `button.secondary*`
- `checkbox.*`, `dropdown.*`
- `keybindingLabel.*`

## Pickers And Notifications

- `quickInput.background`, `quickInput.foreground`, `quickInputTitle.background`
- `quickInputList.focusBackground`, `quickInputList.focusForeground`, `quickInputList.focusIconForeground`
- `pickerGroup.border`, `pickerGroup.foreground`
- `notifications.*`, `notificationCenter.*`, `notificationToast.border`, `notificationLink.foreground`

## Modern VS Code Surfaces

- `commandCenter.*`
- `settings.*`
- `inlineChat.*`, `inlineChatInput.*`, `inlineChatDiff.*`
- `chat.*`
- `testing.*`
- `debugToolBar.*`
- `ports.iconRunningProcessForeground`
- `editorStickyScroll.*`
- `editorInlayHint.*`
- `editorGhostText.foreground`
